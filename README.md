# SH-Install-Scripts

⚠️ **NOTE:** These scripts are exclusively for Ubuntu/Debian based systems. Do not run them on other distributions.

## Installation Instructions

To install any of these tools, copy and paste the corresponding command directly into your terminal:

* **Install Box64**
  ```bash
  wget -qO- https://githubusercontent.com | bash
  ```

* **Install Box86**
  ```bash
  wget -qO- https://githubusercontent.com | bash
  ```

* **Install Pi-Apps**
  ```bash
  wget -qO- https://githubusercontent.com | bash
  ```

* **Install bwfall**
  ```bash
  wget -qO- https://githubusercontent.com | bash
  ```

* **Start Mantis Buddy for Mantis Mouse Pro**
  ```bash
  wget -qO- https://githubusercontent.com | bash
  ```
